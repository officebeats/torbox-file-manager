# Privacy Policy for File Manager for TorBox (Unofficial)

**Effective Date:** July 12, 2026

This Privacy Policy describes how we handle user data in the **File Manager for TorBox (Unofficial)** browser extension (the "Extension"). 

We take user privacy extremely seriously. The Extension is designed to run entirely locally in your browser with no external data collection.

---

## 1. Information Collected and Stored

*   **TorBox API Key:** To authenticate with TorBox, the Extension requires you to input your TorBox API key. This key is stored securely and locally in your browser's local memory (`chrome.storage.sync` with a local fallback). 
*   **Virtual File System (VFS) Structure:** Any folders, subfolders, custom category tags, and file-to-folder mappings you create are stored entirely locally in your browser's storage.
*   **No Personal Data Collection:** The Extension does not collect, record, or store any personal data, email addresses, browsing history, download history, or network traffic.

---

## 2. Data Transmission and Network Use

*   **Direct API Connections:** All network requests made by the Extension are sent directly from your browser to the official TorBox API endpoints (`https://api.torbox.app/*`).
*   **No Third-Party Servers:** We do not operate any third-party servers, databases, or cloud environments. Your API key and file structures are never transmitted to us or any other third party.
*   **Browser-Based Synchronization:** Your local folders and settings are synced across your own devices using your browser's built-in synchronization service (e.g., Google Chrome Sync). This data is encrypted and handled entirely by your browser provider.

---

## 3. Analytics and Tracking

*   The Extension does not use any tracking scripts, third-party analytics (like Google Analytics), or advertisements.

---

## 4. Compliance and Policies

*   The Extension complies with the Chrome Web Store Developer Program Policies, including the User Data Policy.
*   We certify that user data is not sold, rented, or transferred to third parties for marketing, advertising, credit assessment, or lending purposes.
